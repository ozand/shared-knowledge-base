## Archive Policy

Files are moved to archive when:
- The issue/concern has been resolved
- The content is superseded by newer documentation
- The file is project-specific and no longer actively referenced
- The file is a one-time analysis/report

## Note

Archived files are preserved for historical value but should not be considered current documentation.
